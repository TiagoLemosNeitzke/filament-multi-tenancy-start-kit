## Instalação

1. Composer
```bash
composer install
```

2. Migrações
```bash
php artisan migrate
```

3. Seeders
```bash
php artisan db:seed
```

4. ENV
```bash
 cp .env.example .env
```

5. Key
```bash
php artisan key:generate
```
